Comment ça marche

3 étapes, 30 secondes

1

Installez le bookmarklet

Glissez-déposez le bouton violet ci-dessus sur votre barre de favoris Chrome. Si elle n'est pas visible, appuyez sur Ctrl+Shift+B (Windows) ou Cmd+Shift+B (Mac).

2

Ouvrez un produit AliExpress

Naviguez sur AliExpress et ouvrez n'importe quelle page produit. Assurez-vous que la page est bien chargée.

3

Cliquez sur le bookmarklet

Cliquez sur "📦 Scrape AliExpress" dans vos favoris. Un popup apparaît avec toutes les données extraites. Le JSON est copié automatiquement dans votre clipboard.

FAQ

Questions fréquentes

Pourquoi c'est gratuit ?
Le bookmarklet tourne dans VOTRE navigateur. Il n'y a pas de serveur, pas de proxy, pas d'API payante. Vous êtes déjà connecté à AliExpress, donc le script lit simplement les données de la page que vous avez ouverte. Zéro coût.
Est-ce que ça marche avec les protections anti-bot ?
Oui ! Comme le bookmarklet s'exécute dans votre propre session navigateur (avec vos cookies, votre IP), AliExpress voit une navigation normale. Il n'y a aucune requête externe ni comportement suspect.
Ça marche sur quel navigateur ?
Chrome, Edge, Brave, et tout navigateur basé sur Chromium. Firefox fonctionne aussi. Safari peut avoir des limitations avec le clipboard.
Comment envoyer les données vers Product Forge / Apify ?
Après le scrape, cliquez sur "🚀 Send to Apify" dans le popup. Entrez votre token API Apify (gratuit sur console.apify.com). Les données seront envoyées à l'Actor painless_rachis/product-forge-scraper.
Quelle est la différence avec un scraper classique ?
Un scraper classique (Puppeteer, Playwright) coûte des crédits cloud car il lance un navigateur headless. Le bookmarklet utilise votre navigateur existant — c'est comme un copier-coller automatisé. Gratuit, simple, sans infrastructure.
