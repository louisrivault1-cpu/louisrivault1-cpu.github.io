Welcome to Product Forge

Transform AliExpress products into professional Etsy listings

Step 1: Choose Your Method

Paste multiple AliExpress URLs, one per line. Click "Scraper tout" to process all links sequentially.

Method 1: Free

Use Claude's free API to generate Etsy listings

  1. Paste an AliExpress URL
  2. Click "Scraper tout" to extract product data
  3. Copy the generated prompt
  4. Paste it in Claude (claude.ai)
  5. Use the generated JSON to create your Etsy listing

Method 2: API (Faster)

Use your own Claude API key for instant generation

  1. Add your Claude API key in Settings
  2. Paste an AliExpress URL
  3. Click "API Mode" and let Claude generate everything
  4. Get instant Etsy-ready listings

Method 3: Bookmarklet

Scrape products directly from AliExpress

  1. Install the bookmarklet from Settings
  2. Open an AliExpress product page
  3. Click the bookmarklet to send data directly
  4. Product details auto-fill here
